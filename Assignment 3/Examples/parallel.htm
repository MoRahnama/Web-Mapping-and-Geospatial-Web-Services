This WEB SERVICE was developed using PHP language.
It provides one operation:length(input parameter: lat)

...the operation provides THE LENGTH IN KILOMETERS OF ANY PARALLEL IN XML FORMAT, based on the input parameter lat (latitude) in the format +DD.dddd.

The calculation assumes the Earth spherical and with radius of 6,371 km.

Syntax example: http://gaia.gge.unb.ca:8080/gge5403/labs/3/parallel.php?operation=length&lat=45
